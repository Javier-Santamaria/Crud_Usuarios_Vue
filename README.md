# Crud_Usuarios_Vue

Este proyecto es una aplicación de gestión de usuarios desarrollada con **Vue.js 3**, enfocada en demostrar mis habilidades en el ecosistema Front End, manejo de estado y persistencia de datos.

## 💬 Tecnologías Utilizadas

- **Vue.js 3**: Framework principal para el desarrollo de interfaces.
- **Pinia**: Gestión de estado global de la aplicación.
- **Vue Router 4**: Manejo de navegación SPA.
- **SweetAlert2**: Para notificaciones y diálogos de confirmación dinámicos.
- **JSONPlaceholder**: Consumo de API REST para la carga inicial de datos.
- **LocalStorage**: Persistencia de datos local para mantener los cambios del CRUD.

## 📅 Características Principales

- **Gestión Completa (CRUD)**: Crear, Leer, Actualizar y Eliminar usuarios.
- **Validaciones en Tiempo Real**: Control de formatos para nombres, emails y teléfonos.
- **Persistencia**: Los cambios realizados se mantienen incluso después de recargar la página.
- **Diseño UX/UI**: Loader de carga y componentes interactivos.

## 📚 Instalación y Ejecución

1. Clonar el repositorio.
2. Instala las dependencias:
   ```
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```
   npm run dev
   ```
4. Abre en el navegador `http://localhost:5173`.
